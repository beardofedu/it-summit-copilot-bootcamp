# Lab 04 — Advanced Copilot Chat

This lab helps you practice Chat features beyond basic “write this function” prompts.

## Goal

Learn when to use Chat for:
- explanation
- fixing bugs
- generating tests
- scoping with file references
- iterative multi-turn development

## Files in this lab

- `exercises/CHAT_CHALLENGES.md`

## Recommended approach

Work through the challenges from easiest to hardest.

1. Start with explanation
2. Move to tests and refactoring
3. Finish with a multi-turn build workflow

## Tips

- Keep prompts scoped and specific
- Use `#file` references when possible
- Ask follow-up questions instead of trying to get everything in one prompt
- Review generated output like you would review a teammate's pull request

## Good outcomes for this lab

By the end, you should be able to:
- ask Chat to explain unfamiliar code
- guide Chat through a bug fix or refactor
- use slash commands effectively
- iterate through multiple turns to build or improve a feature
